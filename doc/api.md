
# API 说明

## 上层 API

| **序号** | **函数名**                | **说明**                                                                     |
| :------- | :------------------------ | :--------------------------------------------------------------------------- |
| 1        | as60x_set_hand_shake_baud | 设置模块通信波特率，必须设置后才能初始化模块，后期完成自动波特率后弃用该接口 |
| 2        | as60x_init                | 初始化模块                                                                   |
| 3        | as60x_str_fp_to_flash     | 指纹录入函数                                                                 |
| 4        | as60x_search_fp_in_flash  | 指纹搜索函数                                                                 |

## 下层 功能相关 API

| **序号** | **函数名**      | **说明**   |
| :------- | :-------------- | :--------- |
| 1        | fp_vfy_password | 验证密码   |
| 2        | fp_get_image    | 获取图像   |
| 3        | fp_gen_char     | 生成特征值 |
| 4        | fp_search       | 搜索指纹   |
| 5        | fp_reg_model       | 合成模板   |
| 6        | fp_str_char       | 储存模板   |
